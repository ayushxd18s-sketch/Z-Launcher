# Z LAUNCHER

A free, offline-first Minecraft Java Edition launcher for Android.

## Features

- ✅ **No Microsoft account required** — play offline with any username
- ✅ **Custom skin servers** — ely.by and any other authlib-injector compatible server
- ✅ **Custom renderers** — built-in download for Mobile Glues, GL4ES, Zink, and more
- ✅ **Custom backgrounds** — set any image or video from your phone as the launcher background
- ✅ Built on the powerful Zalith Launcher 2 engine
